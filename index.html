Veraball

A free, libre "roll the ball" game

Download For Windows and GNU/Linux. Manual installation for OS X available.

What is Veraball?

Veraball is a free, libre and open source game where you have to roll the ball to reach the goal. Collect coins, and avoid falling out of the level, without running out of time.

The game is made using the libre and open source Godot Engine, supporting both 2D and 3D games.
